### About the Project
This is my first hands-on project in the programming world, developed during the Logic Programming course at [Alura](https://www.alura.com.br). The project is an interactive "Secret Number" game, where users challenge their intuition to guess a value randomly generated by the system. The website features a clean, responsive interface designed with a focus on User Experience (UX).

### Features and User Experience
The game provides real-time feedback, telling the player if the secret number is higher or lower than their current guess. Additionally, I implemented accessibility features using the **ResponsiveVoice** API, which narrates instructions and results out loud. The system also includes state management to track the history of drawn numbers, ensuring that the same number is not repeated until all possibilities within the range have been exhausted.

### Key Learnings:
*   **DOM Manipulation:** I successfully connected JavaScript with HTML, learning how to capture `input` values and dynamically update tag content (`h1`, `p`).
*   **Logic and Algorithms:** I structured the game logic using functions, conditional statements (`if/else`), and recursion for random number generation.
*   **Arrays and Methods:** I used lists to store history and the `.includes()` method for data validation and flow control.
*   **Web Interactivity:** Handling button attributes (such as the `disabled` state) and integrating external libraries via script tags to enhance functionality.
